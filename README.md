# Modern Portfolio Website

A sleek, modern portfolio website built with HTML5, CSS3, and vanilla JavaScript. Features a dark theme with glassmorphism effects, smooth animations, and responsive design.

## 🚀 Features

- **Smart Loading Screen**: Appears only on first visit or after 24 hours
- **Smooth Page Transitions**: Elegant animations when navigating between pages
- **Modern Design**: Dark theme with glassmorphism effects and gradient accents
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: CSS transitions and JavaScript-powered animations
- **Custom Cursor**: Interactive cursor that scales on hover
- **Scroll Animations**: Elements fade in as you scroll
- **Floating Particles**: Subtle particle effects in the background
- **CV Download**: Direct download button for your resume/CV
- **Professional Content**: Showcases projects, skills, and contact information

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with CSS variables, flexbox, grid, and animations
- **JavaScript**: Interactive features and animations
- **Google Fonts**: Space Grotesk and Inter font families

## 🚀 Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Navigate through the different sections using the menu

## 🎭 User Experience Features

### Loading Screen
- **First Visit**: Shows an elegant loading animation with your logo and progress bar
- **Return Visits**: Skips loading screen for faster access (reappears after 24 hours)
- **Smart Detection**: Uses localStorage to remember your visit

### Page Transitions
- **Smooth Navigation**: Elegant fade-out/fade-in animations between pages
- **Loading Overlay**: Subtle loading indicator during page changes
- **Staggered Content**: Elements appear with smooth delays for visual appeal

## 📄 CV Download Setup

To enable the CV download feature:

1. Save your CV as a PDF file
2. Name it exactly: `Praveen_Wijesundara_CV.pdf`
3. Place it in the `cv/` directory
4. The download button on the About page will automatically work

The download button uses the HTML5 `download` attribute for direct file downloads.

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop computers (1200px+)
- Tablets (768px - 1199px)
- Mobile phones (up to 767px)
